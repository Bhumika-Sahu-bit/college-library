# 📚 College Library Web App

A simple and clean **College Library resource website** built with **Node.js, Express, EJS, JavaScript, and CSS**.  
This app allows users to explore semester-wise notes and study materials after signing up and logging in.

🔗 **Live Demo:**  
👉 https://college-library.onrender.com/home :contentReference[oaicite:1]{index=1}

---

## 🌟 Features

✔ User authentication (Signup & Login)  
✔ Library page for BCA semester notes  
✔ Navigation bar with multiple pages  
✔ Rendered UI using EJS templates  
✔ Organized notes sections (Semester-wise)  
✔ Feedback form  
✔ Responsive user interface

## 🛠️ Tech Stack

This project is built using:

- **Node.js** – Backend JavaScript runtime  
- **Express.js** – Routing & server framework  
- **EJS** – Templating engine for dynamic HTML  
- **JavaScript (ES6+)** – Client-side logic  
- **CSS3** – Styling & layout  
- **HTML5** – Frontend structure
